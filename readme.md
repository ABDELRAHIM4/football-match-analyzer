# ⚽ Football Match Analyzer

> AI-Powered Football Match Video Analysis 
## 🎯 Features

- 🎥 **Match Video Analysis** - Upload and analyze any football match video
- 👥 **Player Detection & Tracking** - Real-time player identification and tracking
- ⚡ **Speed & Distance Calculation** - Measure player speeds and running distances
- 🔥 **Heatmaps** - Visualize player positioning on the pitch
- 📊 **Statistical Reports** - Detailed performance metrics and analytics
- 💾 **Export Results** - Download CSV reports and tracked videos

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **YOLOv11** | Player and ball detection |
| **ByteTrack** | Multi-object tracking across frames |
| **Streamlit** | Interactive web interface |
| **OpenCV** | Video processing and manipulation |
| **Plotly** | Interactive charts and graphs |
| **Pandas** | Data analysis and manipulation |
